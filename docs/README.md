# Thesis Documents

This directory contains the documents related to the thesis,
mainly the thesis proposal and the thesis itself.
All documents are typeset with [Typst].

The compiled PDF documents can be found in the [releases tab] of this repository.
See the README in the root of this repository for more information regarding
validating the integrity of the documents.

[Typst]: https://typst.app/
[releases tab]: https://github.com/diogotcorreia/master-thesis/releases
